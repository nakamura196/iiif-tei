# iiif-tei-js

This is a JavaScript library for working with TEI documents in IIIF contexts. It is designed to work with the [IIIF Curation API](http://codh.rois.ac.jp/iiif/curation/).

## Installation

```bash
npm install -g ts-node
```

## Usage

```bash
ts-node src/demo.ts
```
